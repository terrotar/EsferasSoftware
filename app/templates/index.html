Esferas Software API

Esse desafio da empresa Esferas Software foi desenvolvido com a linguagem Python, framework FastAPI, integração com banco de dados(SQLAlchemy e SQLite3), logs, entre outras tecnologias. O projeto consiste basicamente em um CRUD de uma tabela chamada Contatos e é possível visualizar as documentações da API acessando os endpoints "/docs" e "/redoc", uma implementação automática porém totalmente possível customizar fornecida pelo FastAPI.

Para essa aplicação, devido à pequena quantidade de dados a serem armazenados, decidi por escolher o SQLite3 invés do PostgreSQL. Normalmente tenho uma preferência para trabalhar com o PostgreSQL quando alguma coluna de tabela armazenará mais de um valor, pois esse banco de dados permite o tipo de dados Array, facilitando muito as querys e o código. Porém, como a quantidade de dados é pequena, decidi que três e-mails e telefones bastaria.

O projeto foi elaborado com foco no back-end, sendo desenvolvido toda as funcionalidades da API e suas estruturas a princípio. Pela proposta do cargo referente ao desafio, acredito que seria válido implementar também o front-end. Não possuo muita experiência e conhecimentos nessa área, mas utilizei ao máximo as minhas habilidades para realizar. Foi tido como base de referência o próprio site da empresa, como padrões de cores, estrutura das páginas, fontes, entre outros.

Nem todos endpoints estão sendo contemplados na aba Contatos, principalmente por conta do tempo para desenvolver o projeto(quatro dias). Apesar disso, ainda é possível testar a API visualmente um pouco mais do lado do cliente apesar dessa limitação.

Espero que goste da Esferas Software API!
