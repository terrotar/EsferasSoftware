Esferas Software API

Esse desafio da empresa Esferas Software foi desenvolvido com a linguagem Python, framework FastAPI, integração com banco de dados, logs, entre outras tecnologias. O projeto consiste basicamente em um CRUD de uma tabela chamada Contatos e é possível visualizar as documentações da API acessando os endpoints "/docs" e "/redoc", uma implementação automática porém totalmente possível customizar fornecida pelo framework FastAPI.

O projeto foi elaborado com foco no back-end, sendo desenvolvido toda as funcionalidades da API e suas estruturas a princípio. Pela proposta do cargo referente ao desafio, acredito que seria válido implementar também o front-end. Não possuo muita experiência e conhecimentos nessa área, mas utilizei ao máximo as minhas habilidades para realizar. Foi utilizado como referência o próprio site da empresa, como padrões de cores, estrutura das páginas, fontes, entre outros.
